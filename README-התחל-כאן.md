# 🚀 א.ג נגררים - התחלה מהירה

## ⚡ 3 צעדים להפיכה לאפליקציה:

### 1️⃣ צור אייקונים
```
יצר 2 תמונות PNG:
- icon-192.png (192x192 פיקסלים)
- icon-512.png (512x512 פיקסלים)

רקע: אדום (#eb3349)
טקסט: "א.ג" בלבן
```

**המלצה:** שמור את `icon.svg` כ-PNG ושנה גודל

---

### 2️⃣ העלה לשרת

**המלצה: GitHub Pages (חינם!)**

1. צור repository ב-GitHub
2. העלה את הקבצים:
   - ag-nigrar-app.html
   - manifest.json
   - service-worker.js
   - icon-192.png
   - icon-512.png
3. הפעל GitHub Pages
4. קבל קישור!

**חלופות:** Netlify, Vercel, שרת משלך

⚠️ **חשוב:** חייב HTTPS!

---

### 3️⃣ התקן על הטלפון

**Android:**
1. פתח את הקישור
2. לחץ "התקן" / "Add to Home screen"

**iPhone:**
1. פתח ב-Safari
2. כפתור שיתוף 📤
3. "הוסף למסך הבית"
